> **Downloading papers is easy, reading them is hard.**
> **When papers are dense, let the Butler break them down for you.**

Do your "read later" papers end up as "never read"?
Even with translation tools, is it still hard to capture the key points in long academic papers?

Are you facing these literature-reading problems?

- **Pain Point 1**: Too many papers to read. Even with AI, sending papers one by one is inefficient.
- **Pain Point 2**: You forget after reading and must reread repeatedly, wasting time.
- **Pain Point 3**: Papers are too long; even with translation plugins, key ideas are hard to retain.

No worries—your personal AI Butler, `Zotero-AI-Butler`, is here.

It is your 24/7 tireless assistant.
Drop papers into Zotero as usual, and let the Butler handle the heavy lifting.
It reads deeply and turns papers into structured notes so you can quickly understand each one.
Version 3.0 introduces **Image Summary**, generating academic-poster-style visuals with Nano Banana Pro.

## 🎖️ Core Features

1.  **Automated Processing**: Automatically scans newly added PDFs, generates structured Markdown notes with LLMs, and saves them with the paper entry for easy review.
2.  **Multimodal Support**: Supports Base64 PDF upload, robust extraction of formulas/charts/layout, Chinese and English papers, and even image-only PDFs.
3.  **Image Summary**: Summarize a paper with one image. Uses Nano Banana Pro to generate poster-style visuals that help you quickly retain core ideas.
4.  **Mind Map**: Automatically generates a visual mind map of long papers. Supports zoom and export to PNG/OPML for reuse.
5.  **Multi-Paper Review**: Right-click a collection to analyze multiple papers and generate a linked literature review report.
6.  **Immersive Reading**: Built-in sidebar with LaTeX rendering and follow-up Q&A while reading. You can also [pin the AI Butler sidebar](https://github.com/steven-jianhao-li/zotero-AI-Butler?tab=readme-ov-file#8-%E4%BE%A7%E8%BE%B9%E6%A0%8F-ai-%E7%AC%94%E8%AE%B0%E9%A2%84%E8%A7%88) to stay in flow.
7.  **Open Platform**: Fully customizable prompts and multi-provider model APIs. No built-in paid channel for the plugin itself.

## ✨ Feature Introduction

### 1. Intelligent Note Generation and Review

The core mission of AI Butler is to use large language models (like Gemini or OpenAI) to read papers in depth and automatically organize summaries into well-structured Markdown notes, saved under Zotero entries.

- Automatic Note Saving: Generated notes are automatically saved under the corresponding Zotero entries. The note interaction logic is consistent with Zotero's native notes.
  - Note Review: Click on the note entry to view it.
    ![Notes Generated by AI Butler in Zotero](./assets/images/Zotero中由AI管家生成的笔记.png)
  - Side-by-Side Reading: When reading the original paper, you can open the note at the same time and read them together.
    ![Generated Notes Can Be Viewed Side by Side While Reading Papers](./assets/images/生成的笔记在阅读论文时可并排查看.png)

### 2. Three Flexible Ways to Start Tasks

AI Butler provides three ways to fetch and analyze your papers.

- Method 1: Right-Click Trigger (Immediate Processing)
  The most direct interaction mode. When you need immediate analysis:
  - Right-click a paper entry and select "Summon AI Butler for Analysis".
  - The task enters the queue immediately. Click **"Details"** to view live model responses.

![右键菜单中唤醒AI管家进行分析](./assets/images/右键菜单中唤醒AI管家进行分析.gif)

- **Multi-turn re-analysis**: To re-analyze an already summarized paper in more detail (or with another mode), use **"AI Butler Multi-turn Re-read"** from the context menu, then choose **"Multi-turn Concatenation"** or **"Multi-turn Summary"**. This forcibly overwrites existing AI notes.

- Method 2: Auto Cruise (Automatic Processing for New Papers)
  A "set-and-forget" automated workflow.
  - Off by default: to minimize performance impact on Zotero.
  - One-click enable: turn on "Auto-scan New Literature" in "Dashboard" -> "UI Settings".
  - Auto-run: once enabled, when you drag in new papers (e.g., PDFs), Butler waits for Zotero metadata retrieval, then starts analysis automatically.
  - Persistent: setting is saved and remains active after restarting Zotero.

![在仪表盘中开启自动扫描新文献功能后，AI管家会自动处理新添加的论文](./assets/images/在仪表盘中开启自动扫描新文献功能后，AI管家会自动处理新添加的论文.gif)

- Method 3: Batch Processing (Backfill Older Papers)
  Best for old papers in your backlog.
  - Scan unanalyzed papers: in "Dashboard", click "Scan Unanalyzed Papers". Butler finds items without AI notes and groups them by your Zotero structure.
  - Batch select and add: choose papers freely (including folder-level select all) and click "Add to Queue".
  - Background processing: Butler processes them in the background at your configured speed.

![批量扫描未分析论文并添加到任务队列](./assets/images/批量扫描未分析论文并添加到任务队列.gif)

### 3. AI Butler Main Interface

AI Butler has a main page as the central place for task and configuration management. Open it via:
a. Zotero top menu: "Edit" -> "Settings" -> "AI Butler" tab.
b. Right-click any item -> "AI Butler Dashboard".

The dashboard includes four main pages:

- Dashboard: statistics and quick actions.

![仪表盘页面](./assets/images/仪表盘页面.png)

- Task Queue: manage and monitor analysis tasks. Processing follows a producer-consumer model, and this page shows pending, running, completed, and failed tasks.

![任务队列页面](./assets/images/任务队列页面.png)

- Quick Settings: the plugin configuration center:
  - API configuration: supports OpenAI, Gemini, Anthropic, and OpenAI-compatible APIs (including third-party providers like SiliconFlow). After adding keys, click "Test Connection".
    ![API配置与连接测试](./assets/images/API配置与连接测试.png)
  - Task speed: controls papers processed per minute to avoid API limits.
  - PDF processing mode: choose multimodal mode (Base64) or text extraction mode. Base64 is ideal for multimodal models (e.g., Gemini 2.5 Pro) and improves understanding of images/formulas/tables; text mode is for non-multimodal models.
    ![设置任务处理速度与PDF处理方式](./assets/images/设置任务处理速度与PDF处理方式.png)

  - Prompt templates: built-in presets plus custom templates.
    ![提示词模板设置](./assets/images/提示词模板设置.png)

### 6. Prompt Preset Management

AI Butler provides powerful prompt management features:

- **Built-in Presets**: Provides multiple commonly used prompt templates, switchable with one click.
- **Custom Presets**: Supports saving and managing user-defined prompt templates.
- **Real-time Preview**: When editing prompts, you can preview variable substitution effects in real time (like `{{title}}`, `{{authors}}`, etc.).
- **One-Click Reset**: Can restore to system default prompts at any time.

![Prompt Template Settings](./assets/images/提示词模板设置.png)

#### Multi-Turn Dialogue and Summary Modes

AI Butler's new feature—supports multi-turn prompt configuration. Users can customize the prompts for each dialogue turn, including: research background and questions, research methods and techniques, experimental design and results, conclusions and outlook, etc., finally aggregating multi-turn dialogues into one complete summary.

| Mode                 | Description                                      | Token Usage | Applicable Scenarios                  |
| -------------------- | ------------------------------------------------ | ----------- | ------------------------------------- |
| **Single Dialogue**  | One-time generation of complete summary          | Minimal     | Quick reading, time constraints       |
| **Multi-Turn Concat**| Multi-turn in-depth questioning, concatenate all answers | Higher      | Deep study, detailed analysis needed  |
| **Multi-Turn Summary**| AI aggregates and refines after multi-turn dialogue | Medium      | Balance depth and conciseness         |

- **Multi-Turn Prompt Configuration**: Users can customize the prompts for each dialogue turn, including: research background and questions, research methods and techniques, experimental design and results, conclusions and outlook, etc.
- **Final Summary Prompt**: In multi-turn summary mode, you can configure the prompt used by AI for the final aggregation.

![Multi-Turn Dialogue Mode Settings](./assets/images/多轮对话模式设置.png)

### 8. Sidebar AI Note Preview

In Zotero's right-side item panel, AI Butler automatically shows AI notes for the selected paper for quick browsing.

- **Auto detection**: finds and displays AI Butler notes automatically.
- **Styled rendering**: renders notes with polished CSS themes.
- **Collapse/expand**: click the title bar to toggle note area.
- **Resizable height**: drag the bottom handle; size is saved automatically.
- **Font zoom**: use +/- controls in the title bar.
- **Quick generate**: if no AI note exists, generate one in one click.

![侧边栏AI笔记预览](./assets/images/侧边栏AI笔记预览.png)

- **Sidebar follow-up**:
  - **Full follow-up (saved)**: open the chat window for deep follow-up; history is saved to notes.
  - **Quick ask (unsaved)**: temporary question without saving history.

![侧边栏追问](./assets/images/侧边栏追问.png)

- **LaTeX rendering**: sidebar uses KaTeX to render formulas clearly.
  - **Auto-render**: supports `$...$` (inline) and `$$...$$` (block) math.
  - **Horizontal scroll**: long formulas scroll to avoid overflow.
  - **Dark mode**: formula rendering adapts to Zotero light/dark themes.

![侧边栏LaTeX公式渲染](./assets/images/侧边栏LaTeX公式渲染.png)

- **Pin sidebar**: use Zotero's built-in pin behavior to keep the AI Butler panel fixed while browsing papers.
  - **How**: right-click the AI Butler icon on the far right of the sidebar and choose "Pin This Pane". Right-click again and choose "Unpin This Pane" to undo.

    ![固定侧边栏](./assets/images/固定侧边栏.png)

#### 🎨 Contributing New Themes

Welcome to contribute more beautiful themes to AI Butler!

1. Create a new CSS file in the `addon/content/markdown_themes/` directory (e.g., `your-theme-name.css`)
2. In CSS, you can use `#write` or `.markdown-body` as selector prefixes (will auto-adapt)
3. Add the new theme to the `BUILTIN_THEMES` array in `src/modules/themeManager.ts`:
   ```typescript
   { id: "your-theme", name: "Your Theme Name", file: "your-theme-name.css" }
   ```
4. Add an option to the theme dropdown list in `src/modules/views/settings/UiSettingsPage.ts`
5. Add the new theme to `const themes = [` in `src/hooks.ts`
6. Submit a Pull Request!

> Tip: Typora themes can be used directly! The plugin will automatically adapt the `#write` selector.

### 11. Multi-Paper Literature Review Generation

AI Butler supports comprehensive analysis of multiple papers under a collection, automatically generating literature review reports.

**Usage**:

1. Right-click on any **collection**, select **"AI Butler Literature Review"**
2. Set the review name and custom prompts on the configuration page
3. Check the papers to be included in the review (supports multiple PDF selection)
4. Click "Generate Review"

**Generated Content**:

- Create a new "Report" type entry
- Add all selected PDFs as linked attachments to the report
- Attachment naming format: `[First 30 chars of paper title] Original attachment name`
- Automatically generate comprehensive literature review notes

![Literature Review Interface](assets/images/literature-review-config.png)

> 📖 For detailed usage instructions, please refer to [Literature Review Documentation](https://steven-jianhao-li.github.io/zotero-AI-Butler/#/literature-review)

### 12. Mind Map

AI Butler supports automatically generating paper mind maps, visualizing the hierarchical structure of lengthy papers to help you quickly grasp the paper's framework.

**Features**:

- **Auto-Generation**: Automatically generates structured mind maps based on AI analysis, containing core sections like research background, research methods, key results, conclusions, etc.
- **Interactive Operations**: Supports zoom in, zoom out, fit canvas, etc., freely explore paper structure
- **Adjustable Height**: Drag the bottom handle to adjust mind map area height, settings auto-save
- **Export Features**:
  - **PNG Export**: High-definition 2x resolution images, suitable for sharing and archiving
  - **OPML Export**: Standard outline format, can be imported into other mind map tools (like XMind, Mubu, etc.)

**Usage**:

1. Right-click on paper entry, select **"Summon AI Butler to Generate Mind Map"**
2. Wait for AI analysis to complete, mind map automatically displays in sidebar
3. Use toolbar buttons for zooming, exporting, etc.

![Mind Map Feature](./assets/images/思维导图功能.png)

**Custom Settings**:

In "Quick Settings" -> "Mind Map", you can configure:

- **Prompt Template**: Customize mind map structure, not limited to the default four categories
- **Export Path**: Set default export directory (default is desktop)

## 🚀 Installation and Quick Start

### Install the Plugin

1. Open the project's GitHub Releases page.
2. Download the latest `.xpi` release file.
3. In Zotero desktop, go to top menu: "Tools" -> "Add-ons".
4. Drag the downloaded `.xpi` into the add-ons window to install.

### Quick Setup and Usage

1. Right-click any paper -> "AI Butler Dashboard" -> open the "Settings" tab.
2. Configure API: add your API key (Gemini recommended), then click "Test Connection".
3. Enable auto-scan: in "UI Settings", enable "Auto-scan New Literature".

> After this, when you import a new PDF, AI Butler will usually generate notes automatically in about one minute (depends on model speed).
